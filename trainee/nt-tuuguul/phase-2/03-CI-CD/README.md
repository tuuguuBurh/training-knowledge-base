**CI/CD gej yu ve**

## CI/CD => code iig ayulgui tsewer baidlaar production hurgeh procces yum (automat)

## CI
CD ni lint build test shalgaj evderltei aldaatai esgui bol company duremees uur code baina uu geh metiig shalgana 

## CD

CD ni CI hervee amjilttai boloh yum bol shuud prod ruu deploy hiigdeh process yum hunii gariin uildel hereggui auto aar

## CI/CD ursgal

 ├── Build
