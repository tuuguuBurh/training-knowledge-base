Web Fundamentals

Тайлбар

Web Fundamentals гэдэг нь веб хэрхэн ажилладагийг ойлгох үндсэн мэдлэгүүд юм. Үүнд Internet, Client-Server архитектур, HTTP/HTTPS, DNS, Domain Name, Hosting, Browser-ийн ажиллагаа, HTML, CSS, JavaScript, API болон Database зэрэг ойлголтууд ордог.

Жишээлбэл хэрэглэгч URL оруулахад DNS домэйныг IP болгон хөрвүүлж, browser сервер рүү HTTP request илгээнэ. Серверээс ирсэн HTML, CSS, JavaScript-ийг browser render хийж хэрэглэгчид харуулдаг. Эдгээр ойлголтуудыг нийлүүлээд Web Fundamentals гэж нэрлэдэг.

Вэб хэрхэн ажилладаг вэ?

  • Эхлээд хэрэглэгч browser (Chrome, Safari, Firefox гэх мэт) дээр facebook.com шиг domain оруулна.
  • Компьютер domain нэрийг ойлгодоггүй тул DNS ашиглан IP хаяг руу хөрвүүлдэг.
  • Browser сервертэй TCP/TLS холболт үүсгэж аюулгүй холболт тогтооно.
  • Үүний дараа HTTP Request илгээнэ.
  • Сервер хүсэлтийг боловсруулж шаардлагатай бол Database-аас өгөгдөл авч HTTP Response буцаана.
  • Browser HTML-ийг DOM, CSS-ийг CSSOM болгон боловсруулна.
  • Дараа нь Render Tree үүсгэж Layout болон Paint үе шатуудыг хийснээр хуудас хэрэглэгчийн дэлгэц дээр харагдана.
URL → DNS → TCP/TLS → HTTP Request → Server → HTTP Response → Render → User

Үндсэн ойлголтууд

Нэр Утга
HTTP HyperText Transfer Protocol. Browser болон server хооронд өгөгдөл дамжуулах үндсэн протокол. Өгөгдлийг шифрлэдэггүй. GET, POST, PUT, DELETE зэрэг method ашигладаг.
HTTPS HyperText Transfer Protocol Secure. HTTP-ийн аюулгүй хувилбар бөгөөд SSL/TLS ашиглан өгөгдлийг шифрлэж дамжуулдаг.
DNS Domain Name System. Домэйн нэрийг (google.com гэх мэт) IP хаяг руу хөрвүүлдэг систем.
IP Internet Protocol Address. Сүлжээнд холбогдсон төхөөрөмж бүрийг ялган таних хаяг. Жишээ нь: 192.168.1.1
