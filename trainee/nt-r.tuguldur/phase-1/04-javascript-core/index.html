JavaScript ES6 Features

run example даран console.log харна уу

1. var, let, const

ES5 дээр хувьсагчийг var ашиглан зарладаг байсан. ES6 дээр let болон const нэмэгдсэнээр scope болон утга өөрчлөгдөх эсэхийг илүү сайн удирдах боломжтой болсон.

2. Arrow Function

Уламжлалт function бичлэгийг богино бөгөөд уншихад хялбар arrow function ашиглан бичиж болно.

3. Template Literal

String холбоход + ашиглахын оронд template literal ашигласнаар хувьсагчийг текст дотор шууд ашиглах боломжтой.

4. Destructuring

Object эсвэл array доторх утгуудыг шууд хувьсагчид задлан авах боломжтой.

5. Spread Operator

Array болон object-ийн өгөгдлийг хуулж эсвэл нэгтгэхэд ашигладаг.

6. Async / Await

API-аас өгөгдөл авах зэрэг asynchronous үйлдлийг Promise-оос илүү ойлгомжтой байдлаар бичих боломжтой.
